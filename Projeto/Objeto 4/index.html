Considerando os valores da tabela, qual a probabilidade de obter índice de cor = 0 UT. Ao sortear 1 mês entre os 7 que constam nesse quadro, ou seja, P(Cor=0 UH)?


Desenvolvimento do cálculo de encontrar 1 mês entre os 7 e obter mês onde cor = 0:


a) Neste caso o nosso universo é a sequência dos índices de cor:
Cor = 0, 0, 0, 0, 1, 1, 0.


b) Marcando os índices da cor favoráveis da cor ser igual a 0 temos:
Cor = 0, 0, 0, 0, 1, 1, 0.


c) P(Cor=0 UH) =

nº de casos favoráveis de mês com cor = 0 / nº total de meses da tabela
=
5 / 7
= 0,71

Resposta:
Probabilidade de encontrar um mês na tabela acima em que o índice da cor seja 0UH é de 0,71 ou 0,71x100% = 71%
